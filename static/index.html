How It Works

1

Extract

Article content is fetched via Jina Reader with BeautifulSoup fallback, handling paywalls and JS-rendered pages.

2

Classify

Claude AI analyzes the content against Performativ's domain — wealth management, compliance, and regulation.

3

Return

Returns structured JSON with label, confidence, reasoning, and relevance topics — ready for dashboards.

RESTful API

Three endpoints covering health monitoring, classification, and retrieval of recent results.

GET /health Service status
POST /classify Classify article
GET /latest Recent results
Example request 
# Classify a news article
curl -X POST /classify \
  -H "Content-Type: application/json" \
  -d '{"url": "https://finextra.com/..."}'

# Response
{
  "label": "GOOD_NEWS",
  "confidence": 0.87,
  "reasoning": "FiDA regulation...",
  "relevance_topics": ["regulation"],
  "processed_at": "2025-..."
}
