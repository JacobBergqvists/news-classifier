How it works

Article extraction

Fetches content via Jina Reader with BeautifulSoup fallback. Handles paywalls and JS-rendered pages.

AI classification

Claude analyzes content against Performativ's domain — wealth management, compliance, and regulation.

Structured output

Returns label, confidence, reasoning, and topics as validated JSON — ready for dashboards or alerts.

RESTful API

Three endpoints covering health monitoring, classification, and retrieval of recent results.

GET /health Service status
POST /classify Classify article
GET /latest Recent results
Example request 
# Classify a news article
curl -X POST /classify \
  -H "Content-Type: application/json" \
  -d '{"url": "https://finextra.com/..."}'

# Response
{
  "label": "GOOD_NEWS",
  "confidence": 0.87,
  "reasoning": "EU FiDA framework...",
  "relevance_topics": ["regulation"],
  "processed_at": "2025-01-..."
}
