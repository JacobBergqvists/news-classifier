How it works

Article extraction

Fetches and parses content from any news URL using Jina Reader with BeautifulSoup fallback, handling paywalls and JS-rendered pages.

AI classification

Claude analyzes the content against Performativ's domain — wealth management, compliance, regulation — and assigns a structured label.

Structured output

Returns label, confidence score, reasoning, and relevance topics as validated JSON — ready for dashboards, alerts, or further processing.

RESTful API

Three endpoints covering health monitoring, article classification, and retrieval of recent results.

GET /health Service status
POST /classify Classify article
GET /latest Recent results
Example request 
# Classify a news article
curl -X POST /classify \
  -H "Content-Type: application/json" \
  -d '{"url": "https://finextra.com/..."}'

# Response
{
  "label": "GOOD_NEWS",
  "confidence": 0.87,
