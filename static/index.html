How it works

01

Extract

Fetches and parses article content via Jina Reader with BeautifulSoup fallback. Handles paywalls and JS-rendered pages.

02

Classify

Claude AI analyzes the content against Performativ's domain — wealth management, compliance, regulation — and assigns a label.

03

Return

Returns label, confidence, reasoning, and relevance topics as validated JSON — ready for dashboards or further processing.

02 / API

RESTful endpoints.

Three endpoints covering health monitoring, classification, and retrieval of recent results.

GET /health Status
POST /classify Classify
GET /latest History
Example request 
# Classify a news article
curl -X POST /classify \
  -H "Content-Type: application/json" \
  -d '{"url": "https://..."}'

# Response
{
  "label": "GOOD_NEWS",
  "confidence": 0.87,
  "reasoning": "FiDA regulation...",
  "relevance_topics": ["regulation"]
}
